# Instalacja
Aby rozpocząć pracę z SkillJourney, wykonaj poniższe kroki:

Sklonuj repozytorium
git clone [URL-repozytorium]

Zainstaluj zależności:
npm install

Uruchom serwer deweloperski:
npm run dev
